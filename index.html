Clone a Git repo — quick helper

Paste your repository URL below (HTTPS or SSH), then copy the git clone command.

git clone https://github.com/username/repo-name.git
Tip: Use SSH (git@github...) if you have SSH keys configured for passwordless push/pull.
Open your terminal (Ctrl+`) and paste the command to clone the repo.

Example SSH: git@github.com:username/repo-name.git
